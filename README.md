### Run development server

```bash
make run-dev
```

### Run tests

```bash
make test
```

### Update requirements file

```bash
make update
```

### Install packages listed on requirements file

```bash
make install
```

### Test coverage

```bash
coverage run tests
```

### Test coverage report

```bash
coverage report src/*.py
```
